📡 UART Echo Using Polling – dsPIC33

This project demonstrates UART communication using polling mode on the
dsPIC33FJ256GP710A.

The program implements a UART Echo system where the microcontroller receives data through UART and immediately sends the same data back.

🧠 Project Overview

This example shows how to:

    Initialize UART1
    Receive data using polling
    Transmit data using polling
    Echo received characters back to the serial terminal
    The system continuously listens for incoming characters and transmits them back.
